Yan Lab Resources

Download

In collaboration with Sean Williams (Programmer, now at UC Davis), Peter Wonka (ASU) and Stuart Lindsay (ASU), we developed a graphical user interface program for the design of DNA nanostructures and sequences. Please click to download this program and a user manual associated with it.

DNA models created by Tiamat (.dna files): Double crossover (DAE) tile, 4x4 tile, four-helix tile, tetrahedron, and rectangular origami.

Catoons animated using Tiamat (.avi files): 4x4 tile, three-helix bundle, and tetrahedron.

A new version of Tiamat, Tiamat 2.exe, has been developed with new features including RNA design and editing, user-defined constraints, and oxDNA-friendly export options. Please click to download Tiamat 2.exe.
